A out-of-box single site for uploading meal recipes and leaving data like amount of calories, a photo, prep time, instructions, reviews, likes, etc

Current Features:

- simple login
- simple photo upload
- leave comments
- up vote (like) meals
- search by ingredients, prep time, calorie amount
- Shareit installed -- need to add token and a custom template if you wish. 


Roadmap: 

- Proper Brand UI (its' pretty piece meal ... no pun intended)
- Proper Roles ( Admin and Author for edit/delete )
- Proper photo storage and retrival with Amazon S3
- Mobile photo upload and some clean up on the responsive layout for Tablet/Mobile
- Host it.
